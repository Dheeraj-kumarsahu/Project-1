🔹 GENERAL PROJECT QUESTIONS
1. What is the aim of your project?

Answer:
The aim of my project is to computerize the library system for managing books, users, issued books, and late fees efficiently using Python and MySQL.

2. Why did you choose Library Management System?

Answer:
It is a real-life application that helps in managing books, tracking issued books, and calculating fines automatically.

3. Which programming language is used?

Answer:
Python is used as the front-end programming language and MySQL is used as the backend database.

4. Why did you use MySQL as backend?

Answer:
MySQL provides structured data storage, fast data retrieval, data security, and supports relational databases.
